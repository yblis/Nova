Recherche de modèles GGUF sur HuggingFace

Recherchez et téléchargez des modèles de langage GGUF directement depuis HuggingFace. Utilisez les filtres avancés pour trouver le modèle qui correspond à vos besoins.

Guide : Choisir la bonne taille de modèle
0.5-1B : Idéal pour mobile/edge, réponses rapides mais limitées
1-3B : Bon équilibre pour tablettes, tâches simples (Phi, Gemma 2B)
4-9B : Usage général, ordinateurs portables (Qwen 7B, Mistral 7B) ⭐
13-40B : Haute qualité, nécessite GPU puissant ou Apple Silicon
65B+ : Qualité maximale, serveurs ou stations de travail haut de gamme
Quantification recommandée : Q4_K_M (bon compromis) ou Q5_K_M (meilleure qualité)
